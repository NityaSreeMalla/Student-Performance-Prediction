#Student performance Prediction

##Overview
In this project, I built a machine learning model to predict student exam scores based on factors like study hours, attendance, sleep, and other personal and academic attributes.

##What I Did
- Collected and cleaned the dataset
- Handled missing values and converted categorical data into numerical form
- Explored the data using visualizations
- Built and trained two models:
  - Linear Regression
  - Random Forest Regressor
- Compared their performance using R² score
  
##Results
- Linear Regression performed better with an R² score of around 0.69
- Random Forest gave an R² score of around 0.58
  
##Key Learnings
- I understood how different features affect student performance
- Attendance and study-related factors had a strong impact
- The dataset followed mostly linear patterns, which is why Linear Regression worked better
- More complex models are not always better
  
##Visualizations I Used
- Actual vs Predicted values
- Feature importance
- Correlation heatmap
- Distribution of exam scores
  
##Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib & Seaborn
  
##Conclusion
Through this project, I learned how to go from raw data to building and evaluating machine learning models. I also understood the importance of choosing the right model based on the data.
This project helped me strengthen my understanding of machine learning concepts and practical implementation.
